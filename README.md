# Vishakha Karande - AI Engineer Portfolio

This is a modern, interactive personal portfolio website built for an AI & Machine Learning Engineer. It features a stunning 3D interactive particle background, a sleek "glassmorphism" UI, and dedicated sections to showcase accomplishments, projects, and skills.

**[Live Demo](https://your-username.github.io/)** <-- *Replace with your live URL after deploying!*

---

## Features

- **Interactive 3D Hero:** A full-screen 3D particle sphere built with Three.js that reacts to mouse movement.
- **Glassmorphism UI:** A beautiful, blurry glass-like effect on the accomplishment cards.
- **Image Reveal on Hover:** Accomplishment cards reveal a relevant picture when you hover over them.
- **3D Tilt Effect:** Project cards tilt in 3D on hover for a dynamic feel.
- **Fully Responsive:** The design adapts seamlessly to mobile devices, tablets, and desktops.

## Setup and Usage

To run this project on your local machine, follow these steps:

1.  **Clone the repository or download the files.**
    ```
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    ```

2.  **Add Your Assets.** Make sure you have the required images in the `/assets` folder. The required images are:
    - `accomplishment-winner.jpg`
    - `accomplishment-president.jpg`
    - `accomplishment-gssoc.jpg`
    - `accomplishment-hackathon.jpg`

3.  **Open in Browser.** Simply open the `index.html` file in your web browser (like Chrome or Firefox).
    **Note:** An internet connection is required for the Three.js library and Font Awesome icons to load.

---

## Customization

You can easily customize this portfolio with your own information:

- **Text Content:** All text (names, descriptions, project details) can be edited directly in the `index.html` file.
- **Images:** Replace the images in the `/assets` folder with your own. Make sure the filenames match the ones in `index.html`.
- **Styling:** Colors, fonts, and spacing can be changed in the `css/style.css` file. Look for the `:root` variables at the top of the file to easily change the theme colors.
